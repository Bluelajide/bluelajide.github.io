The Mobility Friction Index

A novel 0-100 metric that quantifies how difficult it is to move through Seattle's neighborhoods.

30% Barrier Density + 25% Avg Severity + 25% Permanence + 20% Hotspots = MFI Score

Each component is normalized and weighted to create a comprehensive, actionable score.

Key Findings

01

Severe Inequity Exists

A 20√ó gap exists between the highest friction neighborhood (Industrial District: 96.5) and the lowest (South Lake Union: 4.9). This isn't random‚Äîit reveals systematic under-investment in accessibility infrastructure.

02

Permanence = Design Failure

99% of all barriers are permanent structural issues, not temporary obstacles. This demonstrates that accessibility wasn't designed into the built environment‚Äîit must be retrofitted at significant cost.

03

Clustering Reveals Patterns

698 accessibility hotspots identified using machine learning. The largest cluster contains 5,743 severe barriers in North Beacon Hill, demonstrating zones of historic neglect.

04

Predictable Severity

Our Random Forest model achieved 68.9% accuracy in predicting barrier severity. Barrier type (40%) and geographic location (35%) are the strongest predictors‚Äîgeography is destiny.
